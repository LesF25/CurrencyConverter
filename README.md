
# Оконное приложение на Qt с использование REST API

Необходимо разработать любое оконное приложение с использованием библиотеки QT, которое должно удовлетворять следующим требованиям:

1. Приложение должно содержать как минимум два любых элемента управления;
2. Приложение должно обрабатывать как минимум одно событие от пользователя;
3. Функционал приложения не должен встречаться в работах других студентов.

## Полезная информация 

### Классы:

#### 1. QNetworkAccessManager

Обработчик запросов (отправка и получение сообщений) - менеджер сетевых запросов. Позволяет взаимодействовать с REST API с помощью сетевых методов (get, post, put и т.д.).
Чтобы отправить запрос используется функция QNetworkReply* get(const QNetworkRequest& request). Нам нужно сохранить ответ, чтобы позже, когда он будет получен мы смогли его обработать. Нам нужно знать когда будет получен ответ, поэтому мы подключаемся к сигналу QNetworkReply "void finished()", например к нашему собственному слоту и сохранить ответ в QNetworkReply* m_reply, а затем прочитать данные ответа. Мы можем это сделать с помощью QByteArray result = m_reply->readAll(). Ответ мы получаем в формате json.
Также данный класс предоставляет сигналы для обработки особых случаев таких как authenticationRequired и sslError.

#### 2. QNetworkRequest

Данный класс представляет собой запрос и содержит url, заголовки и другую информацию. Url устанавливается с помощью функции setUrl(const QUrl& url).

#### 3. QNetworkReply

Класс ответа HTTP в Qt. Объект QNetworkReply содержит данные ответа и заголовки ответа.

#### 4. QJson

**QJsonDocument** - это класс, который упаковывает полный документ JSON и может читать и записывать этот документ как из текстового представления в кодировке UTF-8, так и из собственного двоичного формата Qt. Документ JSON можно преобразовать из его текстового представления в QJsonDocument с помощью QJsonDocument::fromJson().toJson() преобразует его обратно в текст.

**QJsonObject** - Используется для представления объекта в типе данных JSON

**QJsonArray** - Используется для представления массива в типе данных JSON

#### 5. QByteArray

QByteArray - удобный инструмент для манипулирования байтовыми данными в Qt. QByteArray - это класс в Qt Framework, который представляет собой массив байтов. Он может использоваться для хранения и обработки бинарных данных, таких как файлы, изображения, звуковые файлы и др.
